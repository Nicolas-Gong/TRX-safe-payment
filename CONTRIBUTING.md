# Contributing to TRX Safe Payment

First off, thank you for considering contributing to TRX Safe Payment!

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct.

## How Can I Contribute?

### Reporting Bugs

- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce.
- State which version of the App and Android you are using.

### Suggesting Enhancements

- Open a GitHub Issue with a clear description of the enhancement.
- Explain why this enhancement would be useful.

### Pull Requests

- Pull requests are always welcome.
- Please ensure your code follows the existing style.
- Reference any related issues in your PR description.

## Development Setup

1. Clone the repository.
2. Open in Android Studio Iguana+.
3. Build and run on an emulator or physical device.

## Security Disclosures

If you find a security vulnerability, please **do not** open a public issue. Instead, report it privately according to the instructions in our [SECURITY.md](SECURITY.md).
